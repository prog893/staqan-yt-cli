# Engagement Commands

Commands for working with comments and captions on YouTube videos.

## list-comments

List comments for a YouTube video.

### Usage

```bash
staqan-yt list-comments <videoId>
```

### Arguments

- `videoId` - YouTube video ID or video URL

### Options

- `--output <format>` - Output format: json, table, csv, text, pretty (default: pretty)
- `-l, --limit <number>` - Limit number of results (default: 20)
- `-s, --sort <order>` - Sort order: `top` or `new` (default: `top`)
- `-v, --verbose` - Enable verbose output with debug information
- `-h, --help` - Show help

### Examples

```bash
# List top comments
staqan-yt list-comments dQw4w9WgXcQ

# List newest comments first
staqan-yt list-comments dQw4w9WgXcQ --sort new

# Get more comments
staqan-yt list-comments dQw4w9WgXcQ --limit 50

# Export to CSV
staqan-yt list-comments dQw4w9WgXcQ --output csv > comments.csv

# Export to JSON
staqan-yt list-comments dQw4w9WgXcQ --output json
```

### Output Fields

- `id` - Comment ID
- `authorDisplayName` - Author name
- `textOriginal` - Comment text
- `likeCount` - Number of likes
- `publishedAt` - Publication date
- `updatedAt` - Last updated date
- `parentId` - Parent comment ID (for replies)

### Sort Orders

- `top` (default) - Sort by likes (most popular first)
- `new` - Sort by publication date (newest first)

### Use Cases

- **Engagement monitoring** - Track audience feedback
- **Sentiment analysis** - Analyze comment sentiment
- **FAQ extraction** - Find common questions
- **Community insights** - Understand audience needs

## list-captions

List all caption tracks for a YouTube video.

### Usage

```bash
staqan-yt list-captions <videoId>
```

### Arguments

- `videoId` - YouTube video ID or video URL

### Options

- `--output <format>` - Output format: json, table, text, pretty (default: pretty)
- `-v, --verbose` - Enable verbose output with debug information
- `-h, --help` - Show help

### Examples

```bash
# List all captions
staqan-yt list-captions dQw4w9WgXcQ

# Export to JSON
staqan-yt list-captions dQw4w9WgXcQ --output json

# Find auto-generated captions
staqan-yt list-captions dQw4w9WgXcQ --output json | \
  jq '.[] | select(.trackKind == "asr")'
```

### Output Fields

- `id` - Caption track ID (used for download)
- `language` - Language code (e.g., "en", "ja")
- `name` - Track name
- `trackKind` - Track type:
  - `standard` - Manual caption track
  - `asr` - Auto-generated (speech recognition)
- `isDraft` - Whether this is a draft

### Caption Types

**Manual captions (`standard`):**
- Created by video owner
- Higher accuracy
- May include timing for emphasis

**Auto-generated captions (`asr`):**
- Generated by YouTube's speech recognition
- Available for most videos
- Variable accuracy

### Related Commands

- `get-caption` - Download caption content

## get-caption

Download caption content to stdout.

### Usage

```bash
staqan-yt get-caption <captionId>
```

### Arguments

- `captionId` - Caption track ID (get from `list-captions`)

### Options

- `--format <format>` - Caption format: srt, vtt, sbv, srv2, ttml, json (default: json)
- `-v, --verbose` - Enable verbose output with debug information
- `-h, --help` - Show help

### Examples

```bash
# Get caption ID first
staqan-yt list-captions dQw4w9WgXcQ

# Download captions (JSON format by default)
staqan-yt get-caption en.dQw4w9WgXcQ

# Download as SRT (subtitle file format)
staqan-yt get-caption en.dQw4w9WgXcQ --format srt > captions.srt

# Download as VTT
staqan-yt get-caption en.dQw4w9WgXcQ --format vtt > captions.vtt

# Download as text
staqan-yt get-caption en.dQw4w9WgXcQ --format json | jq -r '.[].text'
```

### Caption Formats

**Text formats:**
- `json` - Structured JSON with timing (default)
- `srv2` - YouTube's internal format

**Subtitle formats:**
- `srt` - SubRip format (most players)
- `vtt` - WebVTT format (web players)
- `sbv` - YouTube subtitle format
- `ttml` - TTML format (professional)

### Caption ID Format

Caption IDs typically follow the format:
```
<language_code>.<video_id>
```

Examples:
- `en.dQw4w9WgXcQ` - English captions
- `ja.dQw4w9WgXcQ` - Japanese captions
- `en.dQw4w9WgXcQ.3` - Multiple English tracks

### Working with Captions

**Extract plain text:**
```bash
staqan-yt get-caption en.dQw4w9WgXcQ --format json | \
  jq -r '.[].text' > transcript.txt
```

**Create word cloud:**
```bash
staqan-yt get-caption en.dQw4w9WgXcQ --format json | \
  jq -r '.[].text' | \
  tr '[:upper:]' '[:lower:]' | \
  tr -d '[:punct:]' | \
  tr ' ' '\n' | \
  sort | uniq -c | sort -rn
```

**Find mentions:**
```bash
staqan-yt get-caption en.dQw4w9WgXcQ --format json | \
  jq -r '.[].text' | \
  grep -i '@\|twitter\|instagram'
```

## Common Patterns

### Analyze Comment Sentiment

```bash
# Export comments for sentiment analysis
staqan-yt list-comments VIDEO_ID --output json | \
  jq -r '.[].textOriginal' > comments.txt

# Use with sentiment analysis tools
# sentiment_analysis comments.txt
```

### Find Most Liked Comments

```bash
# Sort comments by likes
staqan-yt list-comments VIDEO_ID --output json | \
  jq 'sort_by(.likeCount | tonumber) | reverse | .[0:10]'
```

### Extract Video Transcript

```bash
# Get plain text transcript from captions
staqan-yt list-captions VIDEO_ID --output json | \
  jq -r '.[0].id' | \
  xargs staqan-yt get-caption --format json | \
  jq -r '.[].text' > transcript.txt
```

### Download Multiple Captions

```bash
# Download all available caption tracks
staqan-yt list-captions VIDEO_ID --output json | \
  jq -r '.[].id' | \
  while read caption_id; do
    language=$(echo $caption_id | cut -d. -f1)
    staqan-yt get-caption "$caption_id" --format srt > "${language}.srt"
  done
```

### Caption Search

```bash
# Search for specific terms in captions
staqan-yt get-caption en.VIDEO_ID --format json | \
  jq -r '.[] | select(.text | contains("keyword"))'
```

### Comment Frequency Analysis

```bash
# Find most common words in comments
staqan-yt list-comments VIDEO_ID --output json | \
  jq -r '.[].textOriginal' | \
  tr '[:upper:]' '[:lower:]' | \
  tr -d '[:punct:]' | \
  tr ' ' '\n' | \
  sort | uniq -c | sort -rn | head -n 20
```

## Tips

1. **Use `--limit`** to control number of comments retrieved
2. **Sort by `new`** for latest feedback, `top` for most popular
3. **Export to JSON** for text analysis and NLP
4. **Use SRT format** for subtitle editors
5. **Auto-generated captions** are available for most videos
6. **Manual captions** are more accurate but not always available

## Notes

- **Comments** - Public comments on videos (requires read scope)
- **Captions** - Subtitle tracks (including auto-generated)
- **Data delay** - Comments and captions update in real-time
- **Rate limits** - Respect YouTube API rate limits
